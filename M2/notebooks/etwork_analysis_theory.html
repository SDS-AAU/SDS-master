Introduction

Networks so what?

The basic jargon

  • *Plainly speaking, a network is a system of elements which are connected by some relationship.
  • The whole system is called a network or graph. The elements are commonly referred to as nodes (system theory jargon) or vertices (graph theory jargon) of a graph. The connections are edges or links.
